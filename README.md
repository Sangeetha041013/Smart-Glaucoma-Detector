# Smart-Glaucoma-Detector
An AI-based system to detect glaucoma from retinal fundus images using deep learning. It helps in early diagnosis by providing fast and reliable predictions.

🚀 Features
* Detects glaucoma from eye images
* Provides prediction with confidence score
* Uses deep learning models (ResNet50 & DenseNet121)
* Simple and user-friendly interface
  
🧑‍💻 Tech Stack
* Python
* TensorFlow / Keras
* OpenCV
* NumPy, Pandas
  
⚙️ How It Works
* Upload retinal image
* Image preprocessing
* Model prediction
* Output: Glaucoma / Normal with confidence
  
🧠 Models
* ResNet50
* DenseNet121 (higher accuracy ~98.68%)
   
🎯 Objective
To build an automated system for early glaucoma detection and assist doctors with quick diagnosis. 

📌 Future Scope
Mobile app integration
Multi-disease detection
